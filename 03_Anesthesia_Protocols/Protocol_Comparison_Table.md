# Protocol Comparison Table — rTKA Anesthesia Pathways

| Feature | OFIA + ACB/IPACK | OFIA (Block-Free) | Opioid-Based |
|--------|------------------|-------------------|--------------|
| **Intraoperative Opioids** | None | None | Yes |
| **Regional Blocks** | ACB + IPACK | No | Optional |
| **Ketamine Infusion** | 0.1–0.25 mg/kg/hr | 0.15–0.25 mg/kg/hr | Optional |
| **Dexmedetomidine** | 0.2–0.7 mcg/kg/hr | 0.3–0.7 mcg/kg/hr | Optional |
| **Magnesium Adjunct** | Yes | Yes | Optional |
| **Primary Analgesia Source** | Blocks + multimodal | Periarticular infiltration + multimodal | Opioids + multimodal |
| **Expected Pain in PACU** | Low | Mild–moderate | Low–moderate |
| **Early Ambulation** | Best | Good | Variable |
| **PONV Risk** | Lowest | Low | Highest |
| **Ideal Patients** | Opioid-naïve, elderly, high-PONV | Block refusal/anticoagulation | Chronic opioid users |
| **Workflow Speed** | Moderate (blocks needed) | Fastest | Fast |
| **Rescue Strategy** | Ketamine | Ketamine → opioid if needed | Opioids |

**Evidence Level:** ACB/IPACK superiority based on Level 1A meta-analyses (2022–2024). OFIA components supported by RCTs demonstrating 20–30% opioid reduction.
