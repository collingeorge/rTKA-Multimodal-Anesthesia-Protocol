# rTKA Anesthesia Protocol Comparison Table

Quick reference comparing the three anesthetic pathways for robot-assisted total knee arthroplasty. All protocols reflect ERAS 2024, ASA/APSF 2024, and ASRA 2024-2025 guidelines.

---

## Protocol Selection Guide

| Patient Factor | Protocol 1 (OFIA + Blocks) | Protocol 2 (OFIA Block-Free) | Protocol 3 (Conventional Opioid) |
|---------------|---------------------------|------------------------------|----------------------------------|
| **First-line choice?** | **YES** - Gold standard | If blocks contraindicated | If OFIA contraindicated |
| **Anticoagulation concern** | Must meet ASRA windows | **Any timing** | **Any timing** |
| **OSA (STOP-Bang ≥3)** | **Preferred** | **Preferred** | Avoid if possible |
| **Elderly/frail (>80 years)** | **Preferred** | **Preferred** | Avoid if possible |
| **Chronic opioid use** | **Consider first** | **Consider first** | **Appropriate** |
| **Active psychosis** | OFIA contraindicated | OFIA contraindicated | **Use this pathway** |
| **High-degree AV block** | OFIA contraindicated | OFIA contraindicated | **Use this pathway** |
| **Severe renal (CrCl <30)** | OFIA contraindicated | OFIA contraindicated | **Use this pathway** |
| **Patient refuses blocks** | N/A | **Use this pathway** | Alternative option |
| **Rapid workflow needed** | Requires block time | **15 min faster** | **15 min faster** |

## Contraindications Summary

### Protocol 1 (OFIA + Blocks) - DO NOT USE if:

**OFIA contraindications:**
- Active psychosis (ketamine contraindicated)
- High-degree AV block without pacemaker (dexmedetomidine contraindicated)
- Severe renal impairment CrCl <30 (magnesium contraindicated)

**Block contraindications:**
- Patient refusal
- Active systemic infection/sepsis
- Anticoagulation outside ASRA 2024 windows (see table below)
- Amide local anesthetic allergy

### Protocol 2 (OFIA Block-Free) - DO NOT USE if:

**OFIA contraindications:**
- Active psychosis (ketamine contraindicated)
- High-degree AV block without pacemaker (dexmedetomidine contraindicated)
- Severe renal impairment CrCl <30 (magnesium contraindicated)

**USE THIS if:**
- Anticoagulation preventing blocks
- Patient refuses blocks
- Rapid workflow needed
- Anatomical difficulty for blocks

### Protocol 3 (Conventional Opioid) - USE if:

**OFIA contraindicated** (see above)  
**Patient preference** after counseling  
**Chronic opioid use** >30 MME/day for >3 months  
**Severe anxiety** unresponsive to midazolam  

**AVOID if possible for:**
- Severe OSA (STOP-Bang ≥5)
- Frail elderly (age >80, high delirium risk)
- History of severe PONV
- History of postoperative respiratory failure

## Key Decision Points

### When to Choose Protocol 1 (OFIA + Blocks):
**FIRST-LINE CHOICE** for most patients  
Best pain control (NRS 0-3)  
Lowest PONV (<10%)  
Anticoagulation meets ASRA windows  
Patient accepts blocks after counseling  

### When to Choose Protocol 2 (OFIA Block-Free):
Anticoagulation outside ASRA windows  
Patient refuses blocks  
Rapid workflow needed  
Anatomical difficulty preventing blocks  
Still want OFIA benefits without blocks  

### When to Choose Protocol 3 (Conventional Opioid):
OFIA contraindicated (psychosis, AV block, severe renal disease)  
Chronic opioid user >30 MME/day  
Patient preference after counseling  
Practitioner/attending preference

## Clinical Pearls

### All Protocols Share:
- **Propofol 1.5-2 mg/kg is the universal induction agent** at UW
- **Sevoflurane 0.8-1.2 MAC is the primary maintenance option** (propofol TIVA alternative for high PONV/MH risk)
- **Midazolam 1-2 mg preop** is standard anxiolysis across all pathways
- **Multimodal analgesia essential** regardless of pathway
- **Early mobilization goal** (POD 0 preferred for Protocols 1 & 2, POD 1 realistic for Protocol 3)

### Protocol-Specific Notes:
- **Protocol 1:** Gold standard - regional blocks provide superior analgesia, plan for 12-18 hour block duration
- **Protocol 2:** Higher ketamine doses compensate for lack of regional analgesia; surgeon infiltration is CRITICAL (not optional)
- **Protocol 3:** Enhanced monitoring mandatory for respiratory depression; PONV 3-4x higher; consider blocks even with opioid anesthesia (reduces opioid requirement 30-50%)
