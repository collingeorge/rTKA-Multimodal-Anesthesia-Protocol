# Quick Reference Cards  
Robot-Assisted Total Knee Arthroplasty (rTKA) – 2025

Print → laminate → pocket / anesthesia machine

| Pathway 1 – Preferred                          | OFIA + ACB/IPACK (Opioid-Free + Blocks) |
|------------------------------------------------|------------------------------------------|
| **Pre-op**                                     | • Acetaminophen 1 g PO<br>• Celecoxib 400 mg PO<br>• Dexamethasone 10 mg IV<br>• Scopolamine patch (high PONV risk) |
| **Blocks (Pre-Induction)**                     | • **Midazolam 1–2 mg IV** (anxiolysis before blocks)<br>• ACB: Ropivacaine 0.2–0.25%, 20–30 mL<br>• IPACK: Ropivacaine 0.2–0.25%, 20 mL<br>• Document quad-sparing |
| **Induction**                                  | • Midazolam 0.5–1 mg IV (additional if needed)<br>• Propofol 1.5–2 mg/kg<br>• Ketamine 0.3–0.5 mg/kg<br>• Rocuronium 0.6–1.0 mg/kg |
| **Adjuncts**                                   | • Magnesium sulfate 30–50 mg/kg **(typical 40 mg/kg)** over 15–20 min<br>• Dexmedetomidine 0.2–0.7 mcg/kg/hr **(0.2–0.6 in age >70/frail)** |
| **Maintenance**                                | • **Sevoflurane 0.8–1.2 MAC** OR **Propofol TIVA 75–150 mcg/kg/min**<br>• Ketamine 0.1–0.25 mg/kg/hr<br>• Dexmedetomidine (as above)<br>• **Maintain MAP > 65 mmHg** |
| **Emergence**                                  | • Stop dex 10–15 min prior<br>• Ondansetron 4 mg IV<br>• Sugammadex **2 mg/kg (TOF ≥2)** or **4 mg/kg (deep block)** |
| **PACU Rescue**                                | • Expected pain NRS 0–3/10<br>• Ketamine 0.1–0.15 mg/kg IV q10–15 min (max 3)<br>• Then opioids only if inadequate |

| Pathway 2                                      | OFIA – Block-Free |
|------------------------------------------------|-------------------|
| **Pre-op**                                     | Same as Pathway 1 (no midazolam needed for blocks) |
| **Blocks**                                     | None – document reason (anticoagulation/refusal)<br>Surgeon: periarticular infiltration 100-150 mL ropivacaine 0.2% |
| **Induction**                                  | • Midazolam 1–2 mg IV<br>• Propofol 1.5–2 mg/kg<br>• Ketamine 0.4–0.5 mg/kg (higher dose)<br>• Rocuronium 0.6–1.0 mg/kg |
| **Adjuncts / Maintenance**                     | • Magnesium 30–50 mg/kg (typical 40)<br>• Dexmedetomidine 0.3–0.7 mcg/kg/hr (0.2–0.6 in elderly)<br>• **Sevoflurane 0.8–1.2 MAC** OR **Propofol TIVA 75–150 mcg/kg/min**<br>• Ketamine 0.15–0.25 mg/kg/hr<br>• Optional lidocaine 1–2 mg/kg/hr (max 300 mg)<br>• **MAP > 65 mmHg** |
| **Emergence**                                  | • Stop dex 10–15 min prior<br>• Ondansetron 4 mg IV<br>• Sugammadex PRN |
| **PACU Rescue**                                | • Expected pain NRS 3–6/10<br>• Ketamine 0.1–0.15 mg/kg IV q10–15 min (max 3)<br>• Then opioids only if needed |

| Pathway 3                                      | Conventional Opioid-Inclusive |
|------------------------------------------------|-------------------------------|
| **Pre-op**                                     | • Acetaminophen 1 g PO<br>• Celecoxib 400 mg PO<br>• Dexamethasone 8 mg IV<br>• **Midazolam 1–2 mg IV** (preop anxiolysis) |
| **Induction**                                  | • Midazolam 0.5–1 mg IV (additional if needed)<br>• Propofol 1.5–2 mg/kg<br>• Fentanyl 1–2 mcg/kg OR Hydromorphone 0.2–0.4 mg<br>• Rocuronium 0.6–1.0 mg/kg |
| **Maintenance**                                | • **Sevoflurane 0.8–1.2 MAC** OR **Propofol TIVA 75–150 mcg/kg/min**<br>• Fentanyl boluses PRN |
| **Emergence**                                  | • Ondansetron 4 mg IV<br>• Sugammadex PRN<br>• Monitor closely for sedation/respiratory depression |
| **PACU**                                       | • Expected pain NRS 2–6/10 (variable)<br>• Hydromorphone 0.2 mg IV q10 min PRN<br>• Enhanced respiratory monitoring if OSA/elderly |

### Universal Safety Reminders (all pathways)
- **Midazolam 1–2 mg IV** is standard for anxiolysis (does not compromise OFIA)
- **Sevoflurane 0.8–1.2 MAC** OR **Propofol TIVA 75–150 mcg/kg/min** - both equally valid maintenance choices
- **Choose Propofol TIVA for:** High PONV risk, MH susceptibility, practitioner preference
- **NO gabapentinoids** (ASA/APSF 2024)  
- **Lipid emulsion 20%** immediately available (LAST risk with blocks)
- **Dexmed bradycardia:** Stop if HR <50 + symptoms → **glycopyrrolate 0.2–0.4 mg IV** (preferred over atropine)
- **Max ropivacaine** ~3 mg/kg plain (~210 mg for 70 kg patient)
- **Block duration:** 12–18 hours (transition to oral multimodal as blocks wear off)
- **Mobilization goal:** POD 0 (typically 4–6 hours post-op for Pathways 1 & 2)
