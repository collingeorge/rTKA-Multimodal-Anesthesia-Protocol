# Quick Reference Cards  
Robot-Assisted Total Knee Arthroplasty (rTKA) – 2025

Print → laminate → pocket / anesthesia machine

| Pathway 1 – Preferred                          | OFIA + ACB/IPACK (Opioid-Free + Blocks) |
|------------------------------------------------|------------------------------------------|
| **Pre-op**                                     | • Acetaminophen 1 g PO<br>• Celecoxib 400 mg PO<br>• Dexamethasone 10 mg IV<br>• Scopolamine patch (high PONV risk) |
| **Blocks**                                     | • ACB: Ropivacaine 0.2–0.25%, 20–30 mL<br>• IPACK: Ropivacaine 0.2–0.25%, 20 mL |
| **Induction**                                  | • Midazolam 1–2 mg IV<br>• Propofol 1.5–2 mg/kg<br>• Ketamine 0.3–0.5 mg/kg<br>• Rocuronium 0.6–1.0 mg/kg |
| **Adjuncts**                                   | • Magnesium sulfate 30–50 mg/kg **(typical 40 mg/kg)** over 15–20 min<br>• Dexmedetomidine 0.2–0.7 mcg/kg/hr **(0.2–0.6 in age >70/frail)** |
| **Maintenance**                                | • Sevoflurane 0.8–1.2 MAC or TIVA (propofol 75–150 mcg/kg/min)<br>• Ketamine 0.1–0.25 mg/kg/hr<br>• Dexmedetomidine (as above)<br>• **Maintain MAP > 65 mmHg** |
| **Emergence**                                  | • Stop dex 10–15 min prior<br>• Ondansetron 4 mg IV<br>• Sugammadex **2 mg/kg (TOF ≥2)** or **4 mg/kg (deep block)** |
| **PACU Rescue**                                | Ketamine 0.1–0.15 mg/kg IV q10–15 min (max 3) → opioids only if failure |

| Pathway 2                                      | OFIA – Block-Free |
|------------------------------------------------|-------------------|
| **Pre-op**                                     | Same as Pathway 1 |
| **Blocks**                                     | None – document reason |
| **Induction**                                  | • Midazolam 1–2 mg IV<br>• Propofol 1.5–2 mg/kg<br>• Ketamine 0.4–0.5 mg/kg |
| **Adjuncts / Maintenance**                     | • Magnesium 30–50 mg/kg (typical 40)<br>• Dexmedetomidine 0.3–0.7 mcg/kg/hr (0.2–0.6 in elderly)<br>• Sevoflurane 0.8–1.2 MAC or TIVA<br>• Ketamine 0.15–0.25 mg/kg/hr<br>• Optional lidocaine 1–2 mg/kg/hr<br>• **MAP > 65 mmHg** |
| **Emergence**                                  | • Stop dex 10–15 min prior<br>• Ondansetron 4 mg IV<br>• Sugammadex PRN |
| **PACU Rescue**                                | Ketamine 0.1–0.15 mg/kg IV q10–15 min → opioids only if needed |

| Pathway 3                                      | Conventional Opioid-Inclusive |
|------------------------------------------------|-------------------------------|
| **Pre-op**                                     | • Acetaminophen 1 g PO<br>• Celecoxib 400 mg PO<br>• Dexamethasone 8 mg IV<br>• Midazolam 1–2 mg IV |
| **Induction**                                  | • Midazolam 1–2 mg IV (if not given preop)<br>• Propofol 1.5–2 mg/kg<br>• Fentanyl 1–2 mcg/kg OR Hydromorphone 0.2–0.4 mg |
| **Maintenance**                                | Sevoflurane 0.8–1.2 MAC + fentanyl PRN |
| **Emergence**                                  | • Ondansetron 4 mg IV<br>• Sugammadex PRN |
| **PACU**                                       | Hydromorphone 0.2 mg IV q10 min PRN |

### Universal Safety Reminders (all pathways)
- **Midazolam 1–2 mg IV** is standard for anxiolysis (does not compromise OFIA)
- **Sevoflurane** preferred by many practitioners for ease of titration
- **NO gabapentinoids** (ASA/APSF 2024)  
- Lipid emulsion immediately available (LAST risk)  
- Dexmed: stop if HR <50 + symptoms → glycopyrrolate 0.2–0.4 mg IV  
- Max ropivacaine ~3 mg/kg plain
