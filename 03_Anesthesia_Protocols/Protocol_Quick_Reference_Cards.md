# Quick Reference Cards  
Robot-Assisted Total Knee Arthroplasty (rTKA) – 2025

Print, laminate, or save to phone. All doses assume ideal body weight unless noted.

| Pathway 1 – Preferred                         | OFIA + ACB/IPACK (Opioid-Free + Blocks) |
|------------------------------------------------|------------------------------------------|
| **Pre-op Meds**                                | • Acetaminophen 1 g PO<br>• Celecoxib 400 mg PO<br>• Dexamethasone 10 mg IV<br>• Scopolamine patch (high PONV risk) |
| **Blocks**                                     | • ACB: Ropivacaine 0.2–0.25 %, 20–30 mL<br>• IPACK: Ropivacaine 0.2–0.25 %, 20 mL |
| **Induction**                                  | • Propofol 1.5–2 mg/kg<br>• Ketamine 0.3–0.5 mg/kg<br>• Rocuronium 0.6–1.0 mg/kg |
| **Adjuncts**                                   | • Magnesium sulfate 40 mg/kg over 15–20 min<br>• Dexmedetomidine 0.2–0.7 mcg/kg/h (no bolus or ≤0.5 mcg/kg slow) |
| **Maintenance**                                | • Sevoflurane or TIVA<br>• Ketamine 0.1–0.25 mg/kg/h<br>• Dexmedetomidine 0.2–0.7 mcg/kg/h |
| **Emergence**                                  | • Stop dex 10–15 min prior<br>• Sugammadex 2–4 mg/kg<br>• Ondansetron 4 mg |
| **PACU Rescue**                                | Ketamine 0.1–0.15 mg/kg IV q10–15 min (max 3) → opioids only if failure |

| Pathway 2                                      | OFIA – Block-Free (Surgeon infiltration only) |
|------------------------------------------------|-----------------------------------------------|
| **Pre-op Meds**                                | Same as Pathway 1 |
| **Blocks**                                     | None – document reason |
| **Induction**                                  | • Propofol 1.5–2 mg/kg<br>• Ketamine 0.4–0.5 mg/kg |
| **Adjuncts**                                   | • Magnesium 40 mg/kg<br>• Dexmedetomidine 0.3–0.7 mcg/kg/h<br>• Optional lidocaine 1–2 mg/kg/h |
| **Maintenance**                                | Ketamine 0.15–0.25 mg/kg/h + dexmedetomidine |
| **PACU Rescue**                                | Ketamine 0.1–0.15 mg/kg q10–15 min → opioids only if needed |

| Pathway 3                                      | Conventional Opioid-Inclusive |
|------------------------------------------------|-------------------------------|
| **Pre-op Meds**                                | • Acetaminophen 1 g<br>• Celecoxib 400 mg<br>• Dexamethasone 8 mg |
| **Induction**                                  | • Propofol 1.5–2 mg/kg<br>• Fentanyl 1–2 mcg/kg OR Hydromorphone 0.2–0.4 mg |
| **Maintenance**                                | Sevoflurane + fentanyl PRN |
| **PACU**                                       | Hydromorphone 0.2 mg IV q10 min PRN |

**Safety Reminders (all pathways)**  
• No gabapentinoids (ASA/APSF 2024)  
• Lipid emulsion available (LAST risk)  
• Dexmedetomidine: stop if HR <50 + symptoms  
• Ketamine rescue: max 3 doses before opioid consideration
