Robot-Assisted Total Knee Arthroplasty (rTKA) – 2025

# Pathway 1 — OFIA + ACB/IPACK (Preferred ERAS Pathway)

| Section | Quick Reference |
|--------|-----------------|
| **Pre-op** | • Acetaminophen **1 g PO**<br>• Celecoxib **400 mg PO** (or meloxicam 15 mg)<br>• Dexamethasone **8–10 mg IV**<br>• Scopolamine patch (high PONV risk)<br>• **Avoid gabapentinoids** |
| **Blocks (Pre-Induction)** | • **Midazolam 1–2 mg IV** (anxiolysis)<br>• **ACB:** 20–30 mL ropivacaine 0.2–0.25%<br>• **IPACK:** 20 mL ropivacaine 0.2–0.25%<br>• Max ropivacaine dose: **3 mg/kg (~210 mg for 70 kg)**<br>• **Lipid emulsion 20% available** (LAST preparedness) |
| **Induction** | • Midazolam **0.5–1 mg IV**<br>• Propofol **1.5–2 mg/kg**<br>• Ketamine **0.3–0.5 mg/kg**<br>• Rocuronium **0.6–1.0 mg/kg** |
| **Adjuncts** | • Magnesium **30–50 mg/kg IV** over 15–20 min (**avoid CrCl <30 mL/min**)<br>• Dexmedetomidine **0.2–0.7 mcg/kg/hr** (0.2–0.6 if elderly/frail)<br>• Ketamine **0.1–0.2 mg/kg/hr** |
| **Maintenance** | • **Sevoflurane 0.8–1.2 MAC** OR **Propofol TIVA 75–150 mcg/kg/min**<br>• Maintain **MAP ≥65 mmHg**<br>• Phe 50–100 mcg IV (HR >70); Eph 5–10 mg IV (HR <60) |
| **Emergence** | • Stop dexmedetomidine **10–15 min** before extubation<br>• Stop ketamine at closure<br>• Ondansetron **4 mg IV**<br>• Sugammadex **2 mg/kg** (TOF ≥2) or **4 mg/kg** (deep block) |
| **PACU** | • Expected pain **0–3/10**<br>• Ketamine **0.1–0.15 mg/kg IV q10–15 min** (max 3 doses)<br>• Escalate to opioids **only if inadequate** |

# Pathway 2 — OFIA Block-Free (No ACB/IPACK)

| Section | Quick Reference |
|--------|-----------------|
| **Pre-op** | Same as Pathway 1 (acetaminophen, NSAID, dexamethasone, scopolamine). |
| **Blocks** | None – document reason (**anticoagulation, refusal, anatomy, workflow**).<br>**Surgeon infiltration:** 100–150 mL ropivacaine 0.2% (max dose ~3 mg/kg). |
| **Induction** | • Midazolam **1–2 mg IV**<br>• Propofol **1.5–2 mg/kg**<br>• Ketamine **0.4–0.5 mg/kg** (higher dose compensates for no blocks)<br>• Rocuronium **0.6–1.0 mg/kg** |
| **Adjuncts / Maintenance** | • Magnesium **30–50 mg/kg** (avoid CrCl <30)<br>• Dexmedetomidine **0.3–0.7 mcg/kg/hr** (0.2–0.6 if elderly)<br>• **Sevoflurane 0.8–1.2 MAC** OR **Propofol TIVA 75–150 mcg/kg/min**<br>• Ketamine **0.15–0.25 mg/kg/hr**<br>• Optional: **Lidocaine 1–2 mg/kg/hr** (max 300 mg total dose) |
| **Emergence** | • Stop dex 10–15 min prior<br>• Ondansetron **4 mg IV**<br>• Sugammadex PRN |
| **PACU** | • Expected pain **3–6/10**<br>• Ketamine **0.1–0.15 mg/kg IV q10–15 min** (max 3)<br>• Opioids **only if inadequate response** |

# Pathway 3 — Conventional Opioid-Inclusive Anesthesia

| Section | Quick Reference |
|--------|-----------------|
| **Pre-op** | • Acetaminophen **1 g PO**<br>• Celecoxib **400 mg PO**<br>• Dexamethasone **8–10 mg IV**<br>• **Midazolam 1–2 mg IV** (preop anxiolysis)<br>• Continue home opioids for chronic users |
| **Induction** | • Midazolam **0.5–1 mg IV**<br>• Propofol **1.5–2 mg/kg**<br>• **Fentanyl 1–2 mcg/kg** OR **Hydromorphone 0.2–0.4 mg**<br>• Rocuronium **0.6–1.0 mg/kg** |
| **Maintenance** | • **Sevoflurane 0.8–1.2 MAC** OR **Propofol TIVA**<br>• **Fentanyl 25–50 mcg boluses PRN** for surgical stimulation<br>• Dexmedetomidine **0.2–0.4 mcg/kg/hr** optional (lower range due to opioid use) |
| **Emergence** | • Ondansetron **4 mg IV**<br>• Sugammadex as indicated<br>• **Expect longer emergence (10–20 min)**<br>• Low threshold to delay extubation if somnolent, OSA, elderly |
| **PACU** | • Expected pain **2–6/10** (variable)<br>• Hydromorphone **0.2 mg IV q10 min PRN**<br>• **Enhanced respiratory monitoring** (OSA, elderly, high opioid dose) |

# Universal Safety Reminders (All Pathways)

- **Midazolam 1–2 mg IV** is standard pre-op anxiolysis and aligns with OFIA principles  
- **Propofol induction 1.5–2 mg/kg** is universal  
- **Sevoflurane 0.8–1.2 MAC** OR **Propofol TIVA 75–150 mcg/kg/min** — both valid  
- **No gabapentinoids** (ASA/APSF 2024)  
- **Magnesium:** avoid if **CrCl <30 mL/min**  
- **Dexmedetomidine bradycardia:** treat with **glycopyrrolate 0.2–0.4 mg IV**  
- **Ropivacaine max dose:** ~3 mg/kg (≈210 mg for 70 kg)  
- **LAST preparedness:** 20% lipid emulsion must be available  
- **Block duration:** ~12–18 hours → establish multimodal analgesia before wear-off  
- **Mobilization goal:** POD 0 (typically 4–6 hours for Pathways 1 & 2)  
- **Maintain MAP ≥65 mmHg** (individualize for CAD/cerebrovascular disease)
